# KFK — Lyspunkter · SVG Lokalcamp 2026

Presentasjon med [reveal.js](https://revealjs.com) om KFK-prosjektet (Kredittfaglig kompetansesenter) hos SpareBank 1.

Holdt av Henry Sjøen og Ahmed Abdi på SVG Lokalcamp 2026.

## Vis presentasjonen

Åpne `index.html` direkte i nettleseren, eller se den publiserte versjonen på GitHub Pages.

## GitHub Pages

Presentasjonen publiseres automatisk til GitHub Pages ved push til `main`.

> **Forutsetning:** GitHub Pages må være konfigurert til å bruke **GitHub Actions** som kilde.  
> Gå til *Settings → Pages → Source → GitHub Actions* i repoet.
